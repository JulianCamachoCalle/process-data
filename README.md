# Process Data

Panel logístico y de analytics para gestionar envíos, recojos y leads.

## Stack

- **Frontend**: React + Vite + React Router + React Query
- **Backend**: Vercel Serverless Functions (API routes)
- **Datos**: Google Sheets + Supabase (migración en curso)
- **CRM**: Kommo (integración activa)

## Uso

```bash
npm install
npm run dev
```

## Dashboard

Acceder a `http://localhost:5173` después de iniciar el servidor.
