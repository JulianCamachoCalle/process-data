# Landing placeholders

Use these folders to drop future real assets for the public `/land-page` redesign.

## Suggested files

- `hero/hero-shot.jpg` — main editorial hero image
- `story/package-sequence-01.jpg` — box/package reveal frame
- `coverage/coverage-map-dark.png` — coverage or routing visual
- `testimonials/client-grid.jpg` — customer/social proof collage

Keep filenames stable if you want to wire them into code later without changing folder structure.
